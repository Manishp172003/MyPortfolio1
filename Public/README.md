## 🚀 How to Run the App Locally

This portfolio website is designed using **React 18**, **Vite** and styled with **Tailwind CSS**. 

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Manishp172003/MyPortfolio.git
cd MyPortfolio
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Start Development Server
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:3000` or the port displayed on your console.

### 4️⃣ Build for Production
To output the static build files into the `/dist` directory:
```bash
npm run build
```
